# FV TrackFlow 🌊

A minimalist, keyboard-focused Track Template Manager for REAPER.

Designed for speed and an uninterrupted workflow, FV TrackFlow replaces standard REAPER track template menus with a fast, tag-supported, and visually clean interface.

## ✨ Key Features

* **Smart Search & Auto-Complete:** Search by track name, folder path, or tags.
* **Zero Layout Shift UI:** A stable interface that doesn't jump or distract.
* **Intelligent Routing:** Automatically moves inserted templates into their designated "Target Folders" and sets up FX Sends.
* **Contextual Minimalism:** Buttons morph and vanish based on context. 
* **Database Editor:** A drag-and-drop supported editor to manage your library, tags, and routing options.
