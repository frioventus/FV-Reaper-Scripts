# FV TrackFlow 🌊

A minimalist Track Template Manager for REAPER.

Designed for speed and uninterrupted workflow, FV TrackFlow replaces the standard REAPER track template menus with a blazing-fast, tag-supported, and visually clean interface.

## ✨ Key Features

* **Smart Search & Auto-Complete:** Search by track name, folder path, or tags.
* **Zero Layout Shift UI:** A rock-solid, pixel-perfect interface that never jumps, jitters, or distracts.
* **Intelligent Routing:** Automatically moves inserted templates into their designated "Target Folders" and sets up FX Sends on the fly.
* **Contextual Minimalism:** Buttons morph and vanish based on your context. No wasted space.
* **Powerful Database Editor:** A built-in, drag-and-drop supported editor to manage your library, tags, and routing options seamlessly.

## 🚀 Installation (via ReaPack)

1. Make sure you have [ReaPack](https://reapack.com/) installed in REAPER.
2. Go to **Extensions > ReaPack > Import repositories**.
3. Paste the following repository link:
